Errors

(e) All errors are reported using this format (with the HTTP-status code matching the number)

{ status : statusCode, "msg": "Explains the problem" }

  • (e1) :{ status : 400, "msg": "No persons found with this hobby" }
  • (e2) :{ status : 400, "msg": "No persons found living in this city" }
  • (e3) :{ status : 400, "msg": "One of the phone numbers are already in use." }
  • (e4) :{ status : 400, "msg": "Person you are trying to edit cannot be found" }
  • (e5) :{ status : 400, "msg": "Person with given name could not be found" }
  • (e6) :{ status : 400, "msg": "Multiple people with same name and last name found, cannot proceed with editing" }
  • (e7) :{ status : 400, "msg": "Not all required arguments included" }
  • (e8) :{ status : 400, "msg": "No phones found for person with that name" }
  • (e9) :{ status : 400, "msg": "Phone information missing in body" }
  • (e10) :{ status : 400, "msg": "Person already has a phone with that number" }
  • (e11) :{ status : 400, "msg": "This address is already created" }
  • (e12) :{ status : 400, "msg": "This phone is already created" }
  • (e13) :{ status : 400, "msg": "No address found" }
  • (e14) :{ status : 400, "msg": "No hobby found" }
  • (e15) :{ status : 400, "msg": "No phone found" }
